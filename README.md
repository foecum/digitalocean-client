# digitalocean-client
A simple interface that implements the DigitalOcean APIs

## Actions
- Create Droplet/Create multiple droplet
- List droplets
- Reboot a droplet
- Poweroff a droplet
- Rebuild a droplet
- Rename a droplet
- password reset a droplet
- Delete droplet
- Resize droplet
- get user information
- list regions
